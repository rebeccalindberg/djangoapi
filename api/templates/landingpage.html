Test assignment API

Create/Get/Delete user

http://127.0.0.1:8000/api/user/ GET POST

Gets all users. To create a new user, post the following json:


{
    "username": ""
}

http://127.0.0.1:8000/api/user/id GET DELETE

Replace id by the id of the user, eg. 1.

Create/Get/Delete message

http://127.0.0.1:8000/api/message/ GET POST

Gets all messages. To create a new message, post the following json:


{
    "subject": "",
    "message": "",
    "recipient": null
}

where recipient is the id of a user.

http://127.0.0.1:8000/api/message/id GET DELETE

Replace id by the id of the message, eg. 1.

http://127.0.0.1:8000/api/message/?user=id GET

Replace id by the id of the user, eg. 1.

Gets or deletes all messages sent to one user.

Get messages by date

http://127.0.0.1:8000/api/messagesbydate/?start_date=&end_date= GET

Start_date and end_date should be in this format: 2022-12-14T21:22:51.529568Z. Eg: start_date=2022-12-14T21:22:51.529568Z

This fetches all messages in between the start and end date, ordered by date.

Get latest messages

http://127.0.0.1:8000/api/latestmessages/ GET

Fetches new messages that has been sent since the last time this endpoint was called.

Delete multiple messages

http://127.0.0.1:8000/api/message/delete/?id= DELETE

Ids should be comma separated. Eg: id=1,2,3
